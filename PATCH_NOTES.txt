Jarvis GitHub Release Patch

This patch adds a dedicated GitHub Actions workflow that builds the Jarvis debug APK
and publishes it to the repository Releases page.

Added file:
.github/workflows/android-release.yml

No Android Java source files are changed.
No lambdas are added.
AIDE compatibility is unchanged.

Manual release flow:
GitHub > Actions > Android Release > Run workflow > version: v1.0.0

Tag release flow:
git tag v1.0.1
git push origin v1.0.1

The workflow uses:
- actions/checkout@v5
- actions/setup-java@v5
- actions/upload-artifact@v6
- GitHub CLI gh release create/upload
- GITHUB_TOKEN with contents: write
